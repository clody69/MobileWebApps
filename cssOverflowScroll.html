Overflow Scroll

Try itBrowse Source Code

In this example we demonstrate how to enable scrolling within an element. This requires a combination of two properties: overflow: scroll and -webkit-overflow-scrolling: touch

The CSS code to achieve this is below:

index.html.haml 
.scroll {
  height: 150px;
  overflow-y: scroll;
  -webkit-overflow-scrolling: touch;
  border-top: 1px solid black;
}
